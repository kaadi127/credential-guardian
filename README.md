# 🛡️ Credential Guardian

**Credential Guardian** is a modern, Python-based cybersecurity tool designed to analyze URLs for potential phishing threats. It features a sleek Dark Mode interface and a heuristic scoring system to keep users safe from malicious links.



---

## ✨ Features
* **Real-time Analysis:** Instantly calculates a risk score (0-100) for any URL.
* **Heuristic Engine:** Checks for missing HTTPS, suspicious keywords (e.g., *login, bank, verify*), `@` symbol obfuscation, and URL shorteners.
* **Modern UI:** Built with `CustomTkinter` for a professional, high-tech dark theme.
* **Visual Feedback:** Color-coded results (Green for Safe, Orange for Caution, Red for Dangerous).
